EX 01

Sélecteurs & Spécificité

1.1 — Sélecteurs CSS
  • → Tous les liens dans la navigation .main-nav a
  • → Lien actif .main-nav a.active
  • → Article avec classe featured .post.featured
  • → Premier paragraphe de chaque article article p:first-of-type
  • → Éléments li pairs li:nth-child(even)
1.2 — Spécificité (ordre croissant)
RangSélecteurType
1pÉlément HTML
2.textClasse
3p.textÉlément + Classe
4#mainID

⚠ Si même spécificité, la dernière règle définie gagne. Évitez !important — cela casse la logique de la cascade et complique la maintenance.
EX 02

Box Model

2.1 — Calcul de largeur
content-box 300 + (2×20) + (2×5) = 350px
border-box padding + border inclus dans width = 300px 
/* Reset CSS universel recommandé */
* {
  box-sizing: border-box;
}
2.2 — Centrage horizontal et vertical 
div {
  width: 600px;
  margin: auto;
  height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
}
EX 03

Flexbox

3.1 — Navbar 
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.nav-links {
  display: flex;
  gap: 20px;
}

.cards {
  display: flex;
  gap: 20px;
}

.card {
  display: flex;
  flex-direction: column;
}

.card-link {
  margin-top: auto; /* pousse le lien en bas */
}
EX 04

Grid

4.1 — Grille responsive automatique 
.grid-cards {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}
ℹ auto-fit + minmax() crée une grille responsive sans media queries.
4.2 — Layout avec grid-template-areas 
.page-layout {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

sidebar
main
footer
EX 05

Responsive Design

Media Queries — Approche Mobile First
Tablet ≥ 768px Navigation en ligne, sidebar visible 
@media (min-width: 768px) {
  .nav-menu { flex-direction: row; }
  .sidebar  { display: block; }
}
Desktop ≥ 1024px Mise en page multi-colonnes 
@media (min-width: 1024px) {
  .content { display: flex; }
}
✓ On utilise min-width (et non max-width) pour respecter l'approche mobile-first.

EX 07

Animations

  color: white;
}

Spinner — Animation @keyframes 
@keyframes spin {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

.loader {
  animation: spin 1s linear infinite;
}
← démo interactive
